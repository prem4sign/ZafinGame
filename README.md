# Run For Glory 2026

A browser-based endless runner MVP for promoting a corporate marathon event.

## What It Does

- Uses React, TypeScript, PhaserJS, Vite, and TailwindCSS.
- Lets the runner move continuously while the world scrolls.
- Supports keyboard and tap input for jumping.
- Spawns obstacles and collectibles from the extracted asset pack.
- Tracks distance, energy, lives, collectibles, leaderboard submission, and game over state.

## Run It

Install dependencies and start the Vite dev server:

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

## Assets

Extracted sprites live in `src/assets`. The inventory and Phaser usage notes are documented in `src/assets/ASSET_INVENTORY.md`.
